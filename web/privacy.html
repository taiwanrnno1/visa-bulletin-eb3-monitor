🐾 黑咪快報條款

隱私權政策 / 免責聲明 / 使用條款

生效日期：2026 年 6 月 19 日

本頁說明「黑咪快報」如何設計資料使用方式，以及使用本工具時你需要知道的限制。 使用本工具即表示你了解並同意以下內容；若你不同意，請停止使用本工具。

1. 工具目的

黑咪快報是一個協助使用者閱讀、整理與追蹤公開 Visa Bulletin 資訊的小工具， 目前主要聚焦於 EB-3 表 A、台灣 / All Chargeability Areas 相關排期資訊、 Priority Date 距離估算、歷史走勢整理與分享文字產生。

本工具僅供一般資訊整理與個人追蹤參考，不是政府網站，也不是法律、移民、財務或職涯建議服務。

2. 非官方、非法律或移民建議

黑咪快報與美國國務院、USCIS、NVC、任何美國政府機關、律師事務所、雇主、 招募公司或官方移民單位沒有從屬、授權、背書、贊助或營運關係。

本工具不提供法律意見、移民意見、個案策略、送件建議、資格判斷或任何個人化法律建議。 任何重要移民、工作、搬遷、離職、旅遊、財務或法律決定，都應自行向官方網站確認， 並諮詢合格移民律師或適當專業人士。

3. 官方資料優先

Visa Bulletin 日期、排期前進或倒退、可送件資格、Final Action Date、Dates for Filing、 USCIS 案件狀態、政策與程序都可能隨時變動。

若黑咪快報顯示的內容與官方資料不一致，一律以官方資料為準，包括但不限於 U.S. Department of State Visa Bulletin 與 USCIS 官方頁面。

4. 本機儲存：Priority Date 與 Receipt Number

使用者輸入的 Priority Date 與選填的 USCIS Receipt Number，設計上是儲存在你自己的瀏覽器本機 localStorage 裡，用於下次開啟時顯示與計算。這些資料不會主動送到黑咪快報後台建立個人檔案。

如果你使用共用電腦、工作電腦、公共裝置或他人的手機，其他能使用該裝置或瀏覽器的人可能看得到本機儲存資料。 請只在你信任的裝置上使用儲存功能。

你可以透過清除瀏覽器網站資料、localStorage、重新安裝 PWA、刪除主畫面捷徑或使用無痕模式， 來移除本機資料。因為資料存在你的裝置上，黑咪快報無法替你讀取、修改、復原或刪除這些本機資料。

5. 瀏覽器通知與推播訂閱

若你選擇開啟瀏覽器通知，瀏覽器可能會產生一組推播訂閱資料，黑咪快報可能透過 Cloudflare Workers / KV 或類似服務保存該訂閱資料，用於傳送你同意接收的排期更新通知。

推播訂閱資料通常不包含你的 Priority Date 或 Receipt Number。黑咪快報不會用通知訂閱資料建立移民個案判斷。 你可以在瀏覽器或手機系統設定中關閉通知權限，也可以刪除網站資料。

通知可能因 iOS / Android / 瀏覽器限制、網路、系統省電、第三方服務或技術問題而延遲、遺漏或失敗。 不應依賴通知作為唯一資訊來源。

6. 技術資料與第三方服務

即使黑咪快報不主動收集你的移民追蹤資料，網站託管、瀏覽器、網路、CDN、安全與基礎設施服務商 仍可能自動處理有限技術資料，例如 IP 位址、瀏覽器類型、裝置類型、作業系統、約略地區、存取時間、 頁面請求、錯誤紀錄或安全紀錄。

本工具可能使用或連結到 GitHub Pages、GitHub Actions、Cloudflare Workers / KV、 U.S. Department of State、USCIS 及其他第三方服務。第三方服務依其自己的隱私權政策與條款處理資料。 黑咪快報無法控制第三方網站或服務的準確性、可用性、安全性或資料處理方式。

7. Analytics、廣告與追蹤

目前黑咪快報不使用 Google Analytics、廣告 cookie、行為式廣告追蹤或第三方行銷像素。 若未來加入分析、廣告、登入、雲端同步、Email / LINE 訂閱、AI 功能或付款功能，應先更新本政策。

8. 不出售個人資料

黑咪快報不出售你的個人資料，也不出售你的 Priority Date、Receipt Number、排期資訊、通知訂閱資訊或使用紀錄。

9. 準確性、可用性與估算限制

黑咪快報會盡力整理資訊，但不保證內容永遠即時、正確、完整、安全、不中斷或無錯誤。 日期解析、月份比較、排期前進 / 維持 / 倒退、等待時間估算、歷史圖表、通知內容與分享文字都可能有延遲、 錯誤或誤解。

任何等待時間、距離估算、歷史趨勢或未來推測都只是一般參考，不代表未來一定會照過去趨勢發生。 移民排期可能受法律、政策、名額使用、財政年度、國別需求、政府作業與其他不可控因素影響。

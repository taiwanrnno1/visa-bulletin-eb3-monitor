🐾 通知教學

詳細使用說明

📱 iPhone / iOS

  1. 用 Safari 打開正式網站。
  2. 點分享按鈕，選「加入主畫面」。
  3. 從主畫面打開 EB-3 排期。
  4. 按「開啟瀏覽器通知」，再按允許。
  5. 之後有新月份公告，就會跳手機通知。

如果看到 404，請刪掉舊的主畫面圖示，再重新加入一次。

🤖 Android

  1. 用 Chrome 打開正式網站。
  2. 按「開啟瀏覽器通知」。
  3. 跳出權限視窗時按允許。
  4. 可以從 Chrome 選單加入主畫面，之後更方便打開。
  5. 新月份公告時，手機會收到通知。

如果沒有跳通知，請到 Chrome 網站設定確認通知權限是允許。

使用前請了解： 隱私權政策 / 免責聲明 / 使用條款
